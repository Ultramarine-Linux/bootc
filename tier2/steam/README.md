# Ultramarine w/ Bundled Steam

This is an example of a tier 2 derivation image for Ultramarine bootc.

This derivation of Ultramarine includes the Steam gaming platform pre-installed, alongside
some common gaming libraries and tools, including but not limited to:

- Steam
  - Steam Gamescope integration (SteamOS 3-like)
- Protontricks
- Lutris
- MangoHud
- vkBasalt
- [ES-DE](https://es-de.org/)
- Feral Gamemode
- scx
